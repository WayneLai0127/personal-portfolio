### 心得

繼十月參與 AWS 在台北所舉辦之 aws dev day
後，十二月又參與了 GDG 在台中所舉辦之
devfest，兩場活動都是免費參加，且都是全日制的活動 但此次活動規模與內容相較於 aws
dev day 較為簡單，仔細想想後發現原因可能為 devfest 所隸屬單位雖為
Google，但主辦單位為 GDG，而 GDG 全名為 Google developer group 為 Google
開發者社群，而 aws dev day 則是由 aws
直接主辦，因此兩者規模與內容有所差異，但兩者皆為免費參加，且內容豐富，對於想要了解雲端相關知識的人來說是相當不錯的活動
